## 使用 🐂

该模板仅支持 `pnpm` 包管理器 👉 [安装教程](https://pnpm.io/zh/installation)

1. 安装依赖

```shell
pnpm install
```

2. 开发

```shell
pnpm dev

# 开启 host
pnpm dev:host

# 自动打开浏览器
pnpm dev:open
```

3. 预览

```shell
pnpm preview

# 开启 host
pnpm preview:host

# 自动打开浏览器
pnpm preview:open
```

4. 打包

```shell
pnpm build

pnpm build:debug
```

5. 单元测试

```shell
pnpm test
```

6. 单元测试报告生成

```shell
pnpm coverage
```

7. 类型检查

```shell
pnpm typecheck
```

8. 自动创建

```shell
pnpm auto:create
```

9. 自动移除

```shell
pnpm auto:remove
```

10. 依赖更新

```shell
# 依赖版本更新
pnpm deps:fresh
```

```shell
# 以上命令仅对包信息 package.json 进行写入，需要重新执行包安装命令
pnpm i
```

11. 代码规范校验

```shell
pnpm lint

# 校验时修复
pnpm lint:fix
```
